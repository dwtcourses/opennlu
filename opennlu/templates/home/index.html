OpenNLU


OpenNLU is a front-end interface for Natural Language Understanding (NLU) development, powered by RASA, Tensorflow and PyTorch.


NLU is a subdomain of Natural Language Processing (NLP), which performs the task of Named Entity Recognition and Intent Classification.
The goal is to understand the user's utterance, which is to interpret the intent of the request and extract key information.
In dialogue systems, the NLU component produces a structural representation of the user request, so that subsequent dialogue components can process and perform their NLP tasks.

Build Natural Language Understanding Models with RASA

Rasa NLU is an open-source natural language processing tool for intent classification and entity extraction.
It provides infrastructure and tools necessary for building high-performing, resilient, contextual NLU models.
A comprehensive set of cutting-edge NLU components and customisable development pipeline is available to suit application requirements.

Build Natural Language Understanding Models with Tensorflow

TensorFlow is an end-to-end open source platform for machine learning.
This custom feature in OpenNLU supports the fine-tuning of Joint Intent Classification and Slot-Filling model
based on BERT (Bidirectional Encoder Representations from Transformers) for Natural Language Understanding.
